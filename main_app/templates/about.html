Forage - About Page



Forage is an image-forward 'mood board' app, where users can build and add to their Boards by adding images. The images the user adds to their Boards can either be 'foraged' from other users' images, or they can be images that the user has uploaded themselves.



This web app was built by hvb, ic188002 and hphilpotts using Django framework and postgreSQL, as part of a team project completed during General Assembly London's SEI-66 Online course.
